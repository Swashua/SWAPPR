# Public Static App

This directory is served directly by Express.

- `index.html` is the main notebook browsing and swapping app.
- `login.html` handles sign-in and registration.
- `profile.html` shows the standalone profile view.
- `css/` contains page styles.
- `Images/` contains static image assets.
- `js/pagination.js` is a small tested pagination helper.
- `js/app/` contains the focused browser modules used by `index.html`.
